Règlement Inazuma Eleven SD

Bonjour, voici les règles du tournoi sur Inazuma Eleven SD :
Lors du tournoi, toutes grossièretés envers un adversaire entraîneront une disqualification.
Pour éviter toutes confusions, nous demandons un screen du score obligatoirement à la fin du match.
Suite à un problème que j'ai remarqué, quand vous jouez contre quelqu’un, pendent le match et le chargement, vous ne devriez pas aller dans une autre application (Discord...) sinon vous jouerez contre un bot sauf si vous jouez sur téléphone et que vous utilisez Discord sur PC ou inversement.
Les types de match seront définis sur la page de tournoi. Les matchs devront se jouer au level 50.
Pour voir qui vous devez affronter, regardez sur la page du tournoi. Envoyez un message à votre adversaire.
